# 농촌진흥청, 두부 수율 높고 기계수확 쉬운 신품종 콩 ‘다드림’ 보급

농촌진흥청(청장 이승돈)은 두부 가공 적성과 재배 안정성, 수량성을 모두 갖춘 두부용 콩 ‘다드림’* 현장 보급을 본격 확대한다고 밝혔다. *콩 수량과 두부 수율이 높아‘다 드린다’라는 의미 ‘다드림’은 두부 수율이 283%로 기존 ‘대원콩’(216%)보다 67%포인트(p) 높다. 종자 100알 무게가 30.9g으로 ‘대원콩(27.6g)’보다 커서 동일한 양의 원료로 더 많은 두부를 생산할 수 있다. 식감도 부드러
