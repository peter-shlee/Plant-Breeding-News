# 추석 전 햅쌀 출하 ‘조생종 벼 모내기 연시회’ 개최

추석 명절 전후 늘어나는 햅쌀 수요를 감당하기 위해 조생종 벼를 일찍 심어 수확하는 조기재배가 활발하다. 조기재배를 하면, 농가 소득은 올리고 9월 이후 태풍* 피해를 줄일 수 있으며 다른 작물과 이어짓기도 수월하다. *최근 10년간(2016~2025) 국내 영향 태풍 순별 평균 발생 현황: 8월 상 5건 → 8월 중 7건 → 8월 하 6건 → 9월 상 7건 → 9월 중 3건 → 9월 하 2건 농촌진흥청(청장 이승돈) 국립식량과학원은 4월 30일 충청남도 서천군 현장실증 재배
