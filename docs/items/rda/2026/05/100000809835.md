# 국가 바이오 데이터 가치 제고 범부처 협력 속도 낸다

농촌진흥청(청장 이승돈)은 5월 13, 14일 이틀간 전북특별자치도 전주에서 ‘범부처 생명정보센터협의회 공동연수회’를 열고, 범부처 바이오 데이터 협력 체계 강화에 나선다. 이번 공동연수회에는 농업생명공학정보센터(농촌진흥청 슈퍼컴퓨팅센터 운영), 한국생명공학연구원 국가생명연구자원정보센터, 한국과학기술정보연구원(과학기술정보통신부 산하 기관), 국가보건임상연구 정보서비스(질병관리청 산하 기관), 국립생물자원관(기후에너지환경부 산하 기관), 국립해양생물자원관(해양수산부 산하 기관) 6개 정보센터가 참석한
