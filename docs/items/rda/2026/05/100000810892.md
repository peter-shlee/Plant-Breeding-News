# 충남 공주에서 과수화상병 신규 발생 확인

농촌진흥청(청장 이승돈)은 지금까지 과수화상병이 발생하지 않았던 공주시 의당면에 있는 사과 과수원(이하 과원) 1곳(0.4ha)에서 5월 26일 과수화상병 발생을 처음 확인했다고 밝혔다. 이번 과수화상병 발생은 25일 충청남도농업기술원(이하 충남도원)·공주시농업기술센터(이하 공주센터) 정밀 예찰 과정에서 발견됐다. 나뭇잎과 새로 나온 가지(신초)가 흑갈색으로 변하는 등 전형적인 과수화상병 증상을 보였다. 현재 과수화상병 발생이 확인된 과수원에는 외부인 출입 금지 및 공적 방제 조치가 신속하게 취해졌다.
