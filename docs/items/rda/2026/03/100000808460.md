# “생기 있는 식물 보면 편안해진다” 뇌 활성 변화로 확인

식물이 주는 편안함이 단순한 느낌이 아니라, 실제 뇌 반응에 의한 것임이 확인됐다. 농촌진흥청(청장 이승돈)은 가천대학교 뇌과학연구원과 공동으로 원예식물의 3차원 시각 자극이 인간의 뇌 활성에 미치는 영향을 확인했다고 밝혔다. 연구진은 성인 40명에게 3차원 ‘원예식물 영상’과 ‘인공물(건축물) 영상’을 각각 보여주며, 기능적 자기공명영상(fMRI) 분석법을 통해 뇌 활성 변화를 실시간 측정했다. 이 분석법(fMRI)은 뇌 활동에 따라 달라지는 혈중 산소 농도를 측정하는 뇌 영상 기법이다.
