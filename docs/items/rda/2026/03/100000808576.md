# 농촌진흥청, ‘쌀밥 밥맛 평가’ 표준화 나선다

농촌진흥청(청장 이승돈) 국립식량과학원은 3월 26일 식품자원개발부 가공이용연구동(경기도 수원시)에서 ‘벼 육성 계통* 쌀밥 감각 평가** 공유 공동연수(워크숍)’를 열었다. *벼 육성계통: 품종으로 등록되기 전 시험 재배와 특성 평가를 진행하는 벼 계통으로, 품종 후보 단계의 재료를 의미 **쌀밥 감각평가: 밥을 지은 뒤 외관, 향과 맛, 질감 등을 사람이 직접 평가해 품질을 종합적으로 판단하여 밥맛이 우수한 품종 선발에 활용 이번 행사는 최근 밥맛 좋은 고품질 쌀을 선호하는 소비 경향(트렌드)에 맞춰 밥맛이
