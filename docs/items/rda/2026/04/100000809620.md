# ‘밀’ 진화 과정, 엽록체 유전체 해독으로 밝혔다

농촌진흥청(청장 이승돈)은 국내 대표 밀 품종의 엽록체 유전체 정보를 해독해 ‘밀’의 진화 과정과 모계 기원을 새롭게 밝혔다. 밀은 서로 다른 3개의 야생 밀 조상이 오랜 시간 자연 교배해 만들어진 작물이다. 일반적으로 이들 조상 계통을 에이(A), 비(B), 디(D)로 구분하는데, 먼저 에이(A)와 비(B) 유전체를 가진 야생 밀이 교배해 파스타용 밀(AABB, 4배체)이 생겼다. 이후, 디(D) 유전체를 가진 야생종이 더해지며 현재 밀(AABBDD, 6배체)이 만들어진 것으로 알려져 있다.* *
