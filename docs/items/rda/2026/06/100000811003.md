# ‘무멀칭‧깊이거름주기’로 마늘 재배 기계화 성큼

농촌진흥청(청장 이승돈) 국립농업과학원 성제훈 원장은 6월 5일 경남 창녕에서 열린 ‘무멀칭‧깊이거름주기 마늘 재배 기술 현장 평가회’에 참석해 기술 적용 상황을 점검했다. ‘무멀칭’ 마늘 재배는 비닐 피복 작업을 하지 않아 노동력을 절감하고, 농업용 폐비닐 발생을 줄이며, 밭농업 기계화 편리성을 높이는 기술이다. 여기에 뿌리 가까운 토양층(깊이 25~30cm)에 밑거름을 주는 ‘깊이거름주기’를 접목하면 양분 이용 효율도 높일 수 있다.
